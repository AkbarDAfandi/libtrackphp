# Lib-Track: Sistem Informasi Manajemen Perpustakaan Web-Based

**Lib-Track** adalah aplikasi manajemen perpustakaan (Library Management System) yang dirancang untuk efisiensi pelacakan inventaris buku dan sirkulasi peminjaman. Proyek ini dikembangkan sebagai portofolio *Backend Development* untuk mendemonstrasikan implementasi logika database relasional dan manajemen user.

### 1. Manajemen Inventaris (Inventory Control)
* **CRUD System:** Implementasi *Create, Read, Update, Delete* lengkap untuk data buku.
* **Smart Search:** Algoritma pencarian buku berdasarkan ISBN, Judul, atau Penulis menggunakan *query filtering*.

### 2. Sirkulasi & Tracking
* **Log Peminjaman:** Sistem pencatatan *real-time* untuk status peminjaman (Borrowed/Available).
* **Riwayat Transaksi:** Database menyimpan *history* peminjaman untuk audit dan pelaporan.

### 3. Manajemen User (RBAC)
* **Role-Based Access:** Fitur keamanan untuk membedakan akses antara Administrator (Pustakawan) dan Anggota.
* **Member Management:** Sistem registrasi dan pengelolaan data anggota perpustakaan.
